# Arrays

To create arrays in c++ just add brackets __"[]"__ to variable name and inside the size of the array. Remembering the index begin with 0.

To access or attribute value, use the brackets with the index number.

```c++
#include <iostream>

using namespace std;

int main(){

    int ages[5];

	ages[0] = 2;
	ages[1] = 6;
	ages[2] = 14;
	ages[3] = 9;
	ages[4] = 25;

	system("pause");
	return 0;
}

```
